# star-falling-game
Star Falling Game Collect falling stars to score points:  Yellow stars: 1 point Blue stars: 2 points Green stars: 10 points Purple stars: 100 points (rare!) Orange stars: 500 points (very rare!) White stars: 1000 points (ultra rare!) Catch power-ups for shield or slow time.  Avoid the red ghouls or game over!
